# Glixtron Logo Assets

Created multiple logo variations for different use cases:

## Files Created:
- `logo-transparent.svg` - Original logo with transparent background
- `logo-white-labeled.svg` - White background version for light themes
- `logo-no-bg.svg` - No background circle, just the graphic elements
- `logo-horizontal.svg` - Horizontal layout with text
- `favicon.svg` - Simplified version for favicon use

## Features Added:

### 1. **Transparent Background**
- Removed the dark blue circle background
- Maintains the circular clipping path for clean edges
- Works on any background color

### 2. **White-Labeled Version**
- White background with gray monochrome elements
- Suitable for documentation, invoices, or professional contexts
- Maintains brand recognition while being versatile

### 3. **Multiple Formats**
- **Square**: Original circular format
- **Horizontal**: Icon + text side by side
- **Favicon**: Simplified for small sizes
- **No Background**: Just the graphic elements

### 4. **Scalable Vector Format**
- All logos are in SVG format
- Can be scaled to any size without quality loss
- Easy to convert to PNG, JPG, or other formats as needed

## Usage Recommendations:
- **Website Header**: `logo-horizontal.svg`
- **Favicon**: `favicon.svg`
- **Dark Backgrounds**: `logo-no-bg.svg`
- **Light Backgrounds**: `logo-white-labeled.svg`
- **General Purpose**: `logo-transparent.svg`

## Color Palette:
- Dark Blue: `#0a1628`

- Circuit Purple: `#7c3aed`
- DNA Green: `#10b981`
- White: `#ffffff`
- Gray Scale: `#374151`, `#6b7280`, `#9ca3af`
